## Included
- All Codeforces Div.2 contests held in 2025
- Problems with index A, B, C only
- Official Codeforces contests (non-Gym)
- Each problem appears exactly once
- Once added to the sheet, problems are never removed

## Excluded
- Div.1-only contests
- Educational rounds (unless explicitly Div.2)
- Problems with index D, E, F or higher
- Gym contests or Polygon-only problems

## Problem ID Convention
- Format: <contestId><problemIndex>
- Example: 1952A, 1952B, 1952C

## Ratings
- If a problem has no rating at generation time:
  - rating = null
  - Will be filled during the next manual regeneration

## Data Stability Rule
- The 2025 sheet is static
- Data is regenerated only manually (monthly or after major contests)
